# Управление проектами - База данных с системой безопасности

## Описание
Данный проект демонстрирует реализацию системы безопасности базы данных PostgreSQL для управления проектами.

## Структура базы данных
- 4 таблицы: employees, projects, tasks, time_log
- Система ролей и пользователей
- Триггеры для аудита и контроля целостности
- Журналирование изменений

## Установка и настройка
1. Запустите скрипты в порядке нумерации:
   - 01_create_database.sql
   - 02_create_users_roles.sql
   - 03_grant_permissions.sql
   - 04_create_triggers.sql
   - 05_test_data.sql
